## 快速启动步骤

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问系统
- 地址：http://localhost:3000
- 默认账号：`admin`
- 默认密码：`123456`

如遇问题：
1. 检查Node.js版本 >= 16.0.0
2. 清除缓存：`npm cache clean --force`
3. 重新安装：`rm -rf node_modules && npm install`
4. 查看控制台错误信息
