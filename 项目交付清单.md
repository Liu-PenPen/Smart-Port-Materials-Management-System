## 🛠️ 部署要求

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0 或 yarn >= 1.22.0
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 快速启动
```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问系统
# http://localhost:3000
# 默认账号：admin / 123456
```
